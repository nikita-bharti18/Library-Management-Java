# 📚 Library Management System (Java)

A console-based **Library Management System** developed in **Java** that helps manage books, members, and borrowing records efficiently. The system allows librarians to add, update, search, issue, and return books while maintaining accurate library records through a simple and user-friendly interface.

---

## 🚀 Features

### 📖 Book Management

- Add New Books
- Update Book Information
- Delete Books
- Search Books by Title
- Search Books by Author
- Search Books by ISBN
- Display All Books

---

### 👤 Member Management

- Register New Members
- View Member Details
- Update Member Information
- Delete Members
- Display All Members

---

### 🔄 Book Issue & Return

- Issue Books
- Return Books
- Track Issued Books
- Check Book Availability
- Maintain Borrowing Records

---

### 📊 Library Management

- Display Available Books
- Display Issued Books
- Track Total Books
- Manage Library Inventory
- Generate Library Summary

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/nikita-bharti18/Library-Management-Java.git
```

### Navigate to the Project

```bash
cd Library-Management-Java
```

### Compile the Project

```bash
javac *.java
```

### Run the Application

```bash
java Main
```
